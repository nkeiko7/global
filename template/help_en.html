About MUJI Global Management System（Global CMS）

URL

https://gbcms.muji.com/muji.publisher/manage/login/

Procedure of update / public

[Portal]

  1. Login the above URL.
  2. Click menu of 'Edit Configurations'.
  3. Enter the required menu of 'Edit Configurations'.
  4. Click the 'Preview' button at the bottom of the screen.
  5. Click the 'Deploy' button at the bottom of the screen if there is no problem. (This portal top file will be send to the source tree of Manage Content.)
  6. Click the top of the screen title 'MUJI Global Contents Management System', to return to the Global CMS top page.
  7. Click on the 'Manage Contents'.
  8. Open the 'Country Code(jp,us etc.)' folder on the source tree.
  9. Select the 'index.html'.
  10. When you click the 'Deploy' button at the bottom of the screen, publishing settings dialog opens.
    • 'Deploy Resource:' ... public file "(country code) /index.html"
    • 'Deploy to:' ... public destination
    • 'Staging' ... test environment http://st3gb.muji.com/ (country code) /
      ID: admwtpgj PW: ptwgjmad
    • 'Production' ... release http://www.muji.com/ (country code) /
    • 'Deploy Operaion:' Choose ... Upload
    • 'Deploy on:' ... published date and time settings "YYYYMMDD HH: MM"
  11. if you click the "Book Deployment" button, It will be published in the set date and time.

[Featured content]

  1. The template that is provided separately in the reference, you can create a HTML of featured content.
  2. Put the necessary files(html files, image files, and css files) to the folder that you gave the feature name. And make a .zip file.
  3. Log in from the above URL, and open the Global CMS top page.
  4. Click the top of the screen title 'MUJI Global Contents Management System', to return to the GlobalCMS top page.
  5. Click on the 'Manage Contents'.
  6. Open the 'Your Country Code' folder on the source tree.
  7. Select the 'campaign' folder.
  8. When you click the 'Upload' button at the bottom of the screen, opens the Upload Settings dialog.
    • 'Upload Point:' ··· public destination '(country code) / campaign'
    • 'Upload File:' ... Select the .zip file to be published
  9. Click on the 'Upload'. This files will be send to the source tree of Manage Content.
  10. Select the 'upload folder'.
  11. When you click the 'Deploy' button at the bottom of the screen, publishing settings dialog opens.
    • 'Deploy Resource:' ... public file "(country code) /index.html"
    • 'Deploy to:' ... public destination
    • 'Staging' ... test environment http://st3gb.muji.com/ (country code) /
      ID: admwtpgj PW: ptwgjmad
    • 'Production' ... release http://www.muji.com/ (country code) /
    • 'Deploy Operaion:' ... Choose Upload
    • 'Deploy on:' ... published date and time settings "YYYYMMDD HH: MM"
  12. if you click the 'Book Deployment' button, It will be published in the set date and time.
  13. *If you want to remove the files from the production server
    1. Select the deleted files you want on the 'Manage Contents'.
    2. Click the 'Deploy' button at the bottom of the screen.
    3. Choose 'Delete' menu from the 'Deploy Operaion:' (in the publishing settings dialog) and click the 'Book Deployment'

Function

Edit Configurations (Top Page)

  • This is the setting screen of the portal.
  • Save / Preview / Deploy：Upload function of file　(There is the button at the bottom of the screen)
  • ↑ ↓: display order sorting
  • +: Adding a input area
  • x: Delete

Manage Contents

  • Upload and download files
  • Create Folder
  • Delete files and folders
  • setting to the public time (It is used to publish time)

Manage Deployments

  • History list of upload and deletion.

Manage User Settings

  • User Name：Can not be changed
  • User Password：Change in each
  • please do the time setting of each country based on UTC (example: Japan → UTC + 9 hours)
