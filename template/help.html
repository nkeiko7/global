「MUJI Global Management System（以下GlobalCMS）」について

URL

https://gbcms.muji.com/muji.publisher/manage/login/

※ ID、Passwardは別途お知らせします

更新／公開の手順

[ポータル]

  1. 上記URLからログインし、GlobalCMSトップページ開きます。
  2. リンク「Edit Configurations」をクリックしてください。
  3. 「Edit Configurations」画面にて、更新が必要な項目に入力をします。
  4. 画面下部の「Preview」ボタンをクリックします。「Preview」ページが表示されます。
  5. 「Preview」ページに問題が無ければ、画面下部の「Deploy」ボタンをクリックします。（「Manage Contents」画面のソースツリーにファイルが登録されます。）
  6. 画面上部のタイトル「MUJI Global Contents Management System」をクリックして、GlobalCMSトップページに戻ります。
  7. リンク「Manage Contents」をクリックしてください。
  8. 「Manage Contents」画面にて、ソースツリー上の「（国コード）」フォルダを開きます。
  9. 「index.html」を選択します。
  10. 画面下部の「Deploy」ボタンをクリックすると、公開設定ダイアログが開きます。
    • 「Deploy Resource：」・・・公開ファイル　「（国コード）/index.html」
    • 「Deploy to:」・・・・・・・公開先
    • 「Staging」・・・・・・テスト環境 http://st3gb.muji.com/（国コード）/
      ID: admwtpgj PW: ptwgjmad
    • 「Production」・・・・・リリース http://www.muji.com/（国コード）/
    • 「Deploy Operaion:」・・・・Uploadを選択
    • 「Deploy on：」・・・・・・・公開日時設定「YYYYMMDD HH:MM」
  11. 「Book Deployment」ボタンをクリックします。設定した日時に公開されます。

[特集コンテンツ]

  1. 別途提供されるテンプレートを参考にして、特集コンテンツのHTMLを作成します。
  2. キャンペーン名を付けたフォルダにhtmlファイル、画像ファイル、cssファイルを入れ、.zipファイルにします。
  3. 上記URLからログインし、GlobalCMSトップページ開きます。
  4. 画面上部のタイトル「MUJI Global Contents Management System」をクリックして、GlobalCMSトップページに戻ります。
  5. リンク「Manage Contents」をクリックしてください。
  6. 「Manage Contents」画面にて、「（国コード）」フォルダを開き「feature(campaign)」フォルダを選択します。
  7. 画面下部の「Upload」ボタンをクリックすると、アップロード設定ダイアログが開きます。
    • 「Upload Point：」・・・公開先　「（国コード）/feature(campaign)」
    • 「Upload File：」・・・・・・・公開する.zipファイルを選択
  8. 「Upload」ボタンをクリックします。作成したキャンペーンフォルダが「Manage Contents」画面のソースツリーに登録されます。
  9. 該当のフォルダを選択します。
  10. 画面下部の「Deploy」ボタンをクリックすると、公開設定ダイアログが開きます。
    • 「Deploy Resource：」・・・公開ファイル　「（国コード）/feature(campaign)フォルダ」
    • 「Deploy to:」・・・・・・・公開先
    • 「Staging」・・・・・・テスト環境 http://st3gb.muji.com/（国コード）/
      ID: admwtpgj PW: ptwgjmad
    • 「Production」・・・・・リリース http://www.muji.com/（国コード）/
    • 「Deploy Operaion:」・・・・Uploadを選択
    • 「Deploy on：」・・・・・・・公開日時設定「YYYYMMDD HH:MM」
  11. 「Book Deployment」ボタンをクリックします。設定した日時に公開されます。
  12. ※ファイルを本番サーバから削除したい場合
    1. 「Manage Contents」画面のソースツリーで削除したいファイルを選択します。
    2. 画面下部の「Deploy」ボタンをクリックします。
    3. 公開設定ダイアログの項目「Deploy Operaion:」で、「Delete」を選択し、「Book Deployment」ボタンをクリックします。

機能

Edit Configurations (Top Page)

  • ポータルの設定画面です
  • Save / Preview / Deploy：ファイルのアップロード（画面下にボタンがあります）
  • ↑↓ボタン：表示順の並び替え
  • +ボタン：記入エリアの追加
  • xボタン：削除

Manage Contents

  • ファイルのアップロード、ダウンロード
  • フォルダの作成
  • フォルダ、ファイルの削除
  • 本番への公開時間設定

Manage Deployments

  • ファイルアップロードやファイル削除の履歴一覧表示

Manage User Settings

  • User Name：変更できません
  • User Password：各々で変更をお願いしま
  • Timezone：UTCを基準にして、各国の時間設定を行ってください（例：日本→UTC+9時間）
